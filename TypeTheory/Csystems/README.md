Csystems
========

Code on C-systems, mostly coming from Vladimir Voevodsky.


Contents
--------

The files are given in logical order.
* *prelim.v*
  * elementary arithmetical lemmas
* *lTowers.v*
  * towers and pointed towers
* *ltowers_over.v*
  * towers over an object
* *hSet_ltowers.v*
  * towers of sets
* *lC0systems.v*
  * C0 systems
* *lCsystems.v*
  * C systems
